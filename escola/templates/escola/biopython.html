Biopython é um conjunto de ferramentas disponíveis gratuitamente para computação biológica, escritas em Python por uma equipe internacional de desenvolvedores.

É um esforço colaborativo distribuído para desenvolver bibliotecas e aplicativos Python que atendam às necessidades do trabalho atual e futuro em bioinformática. O código-fonte é disponibilizado sob a Licença Biopython, que é extremamente liberal e compatível com quase todas as licenças do mundo.

Biopython segue as convenções usadas pela linguagem de programação Python para tornar mais fácil para os usuários familiarizados com a linguagem. Por exemplo, os objetos Seq e SeqRecord podem ser manipulados via fatiamento, de maneira semelhante às strings e listas do Python.

Um conceito central em Biopython é a sequência biológica, e isso é representado pela classe Seq.

Anotação sequencial

OA classe SeqRecord descreve seqüências, juntamente com informações como nome, descrição e recursos na forma de objetos SeqFeature. Cada objeto SeqFeature especifica o tipo do recurso e sua localização. Os tipos de recursos podem ser "gene", "CDS" (sequência de codificação), "zona de repetição", "parâmetro móvel" ou outros, e a posição dos recursos na sequência pode ser exata ou aproximada.

Input and Output

O Biopython pode ler e gravar em vários formatos de sequência comuns, incluindo FASTA, FASTQ, GenBank, Clustal, PHYLIP e NEXUS. Ao ler arquivos, informações descritivas no arquivo são usadas para preencher os membros de classes Biopython, como SeqRecord. Isso permite que registros de um formato de arquivo sejam convertidos em outros.

Acessando bancos de dados online

Por meio do módulo Bio.Entrez, os usuários do Biopython podem baixar dados biológicos de bancos de dados do NCBI. Cada uma das funções fornecidas pelo mecanismo de busca Entrez está disponível através das funções deste módulo, incluindo a busca e o download de registros.

Filogenia

O módulo Bio.Phylo fornece ferramentas para trabalhar e visualizar árvores filogenéticas. Uma variedade de formatos de arquivo é suportada para leitura e gravação, incluindo Newick, NEXUS e phyloXML. Manipulações e travessias de árvore comuns são suportadas através dos objetos Tree e Clade. Os exemplos incluem a conversão e o agrupamento de arquivos de árvores, a extração de subconjuntos de uma árvore, a alteração da raiz de uma árvore e a análise de recursos de ramificação, como comprimento ou pontuação.

Sistema de Templates

O Django tem uma linguagem de templates poderosa, extensível e amigável. Com ela você pode separar design, conteúdo e código em Python.

Diagrama Genômico

O módulo GenomeDiagram fornece métodos de visualização de sequências dentro do Biopython.Sequências podem ser desenhadas de forma linear ou circular , e muitos formatos de saída são suportados, incluindo PDF e PNG. Os diagramas são criados ao criar faixas e adicionar recursos de sequência a essas faixas. Fazendo um loop sobre os recursos de uma sequência e usando seus atributos para decidir se e como eles são adicionados às trilhas do diagrama, pode-se exercer muito controle sobre a aparência do diagrama final. Os links cruzados podem ser traçados entre trilhas diferentes, permitindo comparar várias seqüências em um único diagrama.

Estrutura Macromolecular

O módulo Bio.PDB pode carregar estruturas moleculares de arquivos PDB e mmCIF e foi adicionado ao Biopython em 2003. O objeto Estrutura é central para este módulo e organiza a estrutura macromolecular de maneira hierárquica: Os objetos Estrutura contêm objetos Modelo que contêm objetos Cadeia que contêm objetos Resíduo que contêm objetos Atom. Os resíduos e átomos desordenados recebem suas próprias classes, DisorderedResidue e DisorderedAtom, que descrevem suas posições incertas.

Genética Populacional

O módulo Bio.PopGen adiciona suporte ao Biopython para Genepop, um pacote de software para análise estatística da genética de populações.Isto permite análises do equilíbrio de Hardy-Weinberg, desequilíbrio de ligação e outras características das frequências alélicas de uma população.

Ferramentas de linha de comando

Muitos dos módulos do Biopython contêm wrappers de linha de comando para ferramentas comumente usadas, permitindo que essas ferramentas sejam usadas dentro do Biopython. Esses wrappers incluem BLAST, Clustal, PhyML, EMBOSS e SAMtools. Os usuários podem criar uma subclasse de uma classe wrapper genérica para adicionar suporte a qualquer outra ferramenta de linha de comando.
