Tutorial Biopython

O Biopython é um conjunto de ferramentas gratuitamente disponibilizadas para computação biológica. Estas ferramentas são desenvolvidas em Python por uma equipe internacional de desenvolvedores.

É um esforço colaborativo para desenvolver bibliotecas e aplicativos Python que atendam às necessidades do trabalho atual e futuro da bioinformática. O código-fonte é disponibilizado sob a Licença Biopython, que é extremamente liberal e compatível com quase todas as licenças do mundo.

O site do Biopython fornece um recurso on-line para módulos, scripts e links da Web para desenvolvedores de software baseado em Python para uso e pesquisa de bioinformática.

Basicamente, o objetivo do Biopython é facilitar ao máximo o uso do Python para bioinformática, criando módulos e classes reutilizáveis ​​e de alta qualidade. Os recursos do Biopython incluem analisadores ou 'parsers' para vários formatos de arquivos utilizados em análises de Bioinformática (BLAST, Clustalw, FASTA, Genbank, ...), acesso a serviços online (NCBI, Expasy, ...), interfaces para programas (Clustalw, DSSP, MSMS ...), uma classe de sequência padrão, vários módulos de 'clustering', uma estrutura de dados de árvore KD etc. e suas documentações.

Vamos analisar um arquivo tipo fasta e responder algumas perguntas:

  • Quantas sequências estão armazenadas no arquivo?
  • Quais são os identificadores (IDs) e os comprimentos de cada uma das sequências armazenadas no arquivo?
  • Selecionar seqüências e calcular qual é o comprimento médio dessas seqüências.
  • Calcular e imprimir a porcentagem de GC em cada uma das seqüências.
  • Criar um arquivo fasta com as as seqüências selecionadas em passos anteriores.
